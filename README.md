# python 爬虫学习示例
> 本项目仅共学习使用，不做任何商业用途

> python版本：3.7

> 由于本人技术限制，代码中存在不完美，后面的学习中会积极优化


### 1、 包request
- 封装ip代理池（免费（不稳定） + 阿布云代理）
- 封装基础的请求信息

### 2、 包utils
- string_utils.py 封装常用字符串工具
- sql_utils.py 中配置sql相关链接

### 3、 包web
> 网站类爬虫存放处
#### 3.1 拉钩网站
- 启动文件
    - crawler_handler.py
- 实现功能
    - 1、爬取岗位信息
    - 2、爬取城市信息
    - 3、爬取城市对应的岗位信息
> 注: 
> - 爬取过程中会出现异常（由于反爬虫机制导致），出现异常后目前是自动重试两次
> - 超出两次放弃爬取，会存放到异常字典中，最后输出到控制台
